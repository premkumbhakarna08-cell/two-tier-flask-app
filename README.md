## Two Tier Flask Application (Frontend + Backend)
## Author: Prem Kumbhakarna

## Project Overview
This project display a **Two Tier Web Application** using **Flask**.
- Frontend: Flask UI
- Backend: Flask REST API
- Containerized using Docker
- Managed using Docker Compose
- CI/CD implemented using Jenkins
- Images pushed to Docker Hub

## Architecture
Frontend communicates with Backend using REST API.
Docker Compose handles service communication.
